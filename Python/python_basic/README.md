## 자료형

- 프로그램의 기본이자 핵심 단위

### 숫자형
- Number
- 숫자형에는 정수형과 실수형이 있다

#### 정수형

- integer, int

#### 실수형

- Floating Number, float

#### 연산자

- / 나누기
- * 곱하기
- ** 거듭제곱
- // 몫
- % 나머지

### 문자열

- 한글자: character문자열
- 두글자: string 문자열
- 홑따옴표와 쌍따옴표
- Escape Character
    - \n 문자열안에서 줄을 바꿀 때 사용
    - \t 문자열 사이에 탭 간격을 줄 때 사용
    - \\ 문자\를 그대로 표현할 때 사용
    - \' 작은따옴표'를 그대로 표현할 때 사용
    - \" 큰따옴표"를 그대로 표현할 때 사용
    - \r 캐리지 리턴(줄 바꿈 문자, 현재 커서를 가장 앞으로 이동)
    - \f 폼 피드(줄 바꿈 문자, 현재 커서를 다음 줄로 이동)
    - \a 벨 소리(출력할 때 PC 스피커에서 '삑'소리가 난다)
    - \b 백스페이스
    - \000 널문자

#### 문자열의 연산

- 문자열은 + 하면 이어 붙인다
- *하면 곱한만큼 반복
- 인덱싱
- 슬라이싱

#### 포매팅

- 첫번째 방법
        - number = "five"; print("I eat %s apples. So I sicked two days" %number)
        - %s 문자열넣기
        - %d 숫자 넣기
        - ()로 2개 이상의 값 넣기
        - %10s는 전체길이가 10개인 문자열 공간에서 대입되는 값을 오른쪽으로 정렬
        - 반대쪽ㄱ인 왼쪽정렬은 %-10s
        - 소수점을 네 번째 자리까지 나타내고 싶은경우 %0.4f
        - 소수점을 네 번째 자리까지 정렬하고 오른쪽으로 정렬 %10.4f

- 두번쨰 방법
        - number = "five"; day =2
        - print("I eat {0} apples. So I sicked {1} days".format(number,day))
        - {} 안에 < > ^ = ! 등으로 정렬 띄우기 채우기 등이 가능

- 세번째 방법 f-string
        - print(f"I eat {number} apples. So I sicked {day} days")
        - 두번쨰 방법과 마찬가지로 {} 안에 기호를 넣어서 제어 가능

#### 문자열 관련 함수

- len() 문자열의 길이를 구하는 함수
- count() 문자열 중 해당 문자의 갯수를 세는 함수
- upper() 대문자로 변환
- lower() 소문자로 변환
- capitalize() 첫번째 문자를 대문자로 바꿈
- find() 문자열 중 해당 문자가 처음으로 나온 위치를 반환
- index() find 함수와 같으나 존재하지 않는 문자를 찾으면 오류 발생
- join() 문자열 삽입
- strip() 공백을 지움
- rstrip,lstrip 각각 좌 우 측 공백 지우기
- replace() 문자열 안의 특정한 값을 다른 값으로 치환하는 함수
- split() 문자열을 나누는 함수
- title() 분리된 문자열 마다 첫번쨰 문자를 대문자로 바꿈

### 참 거짓형(bool 자료형)

- 문자열 ""같이 값이 없는것만 거짓
- 숫자형 0 만 거짓
- 리스트 [] 같이 리스트가 비어있는것이 거짓
- 튜플 딕셔너리도 리스트와 마찬가지

### 데이터 형변환

- int() str() bool() 등으로 변환

### 리스트

- []대괄호로 묶어주어 만든다
- CRUD (creat, read, update, delete)
- a = [1,2,3]
- 인덱싱
- 슬라이싱
- 리스트의 연산은 문자열의 연산과 똑같다
- a[1] = 20 수정하기
- del a[2] 지우기

#### 리스트 관련 함수

- append() 요소추가
- sort() 정렬하기
- reverse() 뒤집기
- index() 리스트안의 해당 요소의 위치를 반환해주는 함수
- insert(a,b) 리스트 a번째 위치에 b를 삽입하는 함수
- remove() 리스트에서 첫번쨰로 나오는 요소를 삭제하는 함수
- pop() 리스트의 맨 마지막 요소를 보여주고 그 요소는 삭제한다
- count() 리스트안에 요소의 갯수를 세는 함수

### 튜플

- 리스트와 99% 같다
- 리스트는 []로 둘러싸지만 튜플은 ()로 둘러싼다
- 튜플은 리스트와 다르게 수정할수 없다

### 딕셔너리

- 딕셔느리는 {} 중괄호를 써서 만든다
- ex) dic = {'name':'pey','phone':'0119993323','apple':'사과, 능금, 등등'}
- 딕셔너리는 { key : value }의 한 쌍이다

### 집합 자료형

- set() 형식
- union() 합집합 |
- intersection() 교집합 &
- difference() 차집합 -

### 사용자 입력

- input() 함수
- input은 문자열을 출력
