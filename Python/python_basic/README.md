#### 포매팅

- 첫번째 방법
        - number = "five"; print("I eat %s apples. So I sicked two days" %number)
        - %s 문자열넣기
        - %d 숫자 넣기
        - ()로 2개 이상의 값 넣기
        - %10s는 전체길이가 10개인 문자열 공간에서 대입되는 값을 오른쪽으로 정렬
        - 반대쪽ㄱ인 왼쪽정렬은 %-10s
        - 소수점을 네 번째 자리까지 나타내고 싶은경우 %0.4f
        - 소수점을 네 번째 자리까지 정렬하고 오른쪽으로 정렬 %10.4f

- 두번쨰 방법
        - number = "five"; day =2
        - print("I eat {0} apples. So I sicked {1} days".format(number,day))
        - {} 안에 < > ^ = ! 등으로 정렬 띄우기 채우기 등이 가능

- 세번째 방법 f-string
        - print(f"I eat {number} apples. So I sicked {day} days")
        - 두번쨰 방법과 마찬가지로 {} 안에 기호를 넣어서 제어 가능

상속을 하는 이유

보통 기존클래스를 변경하거나 기능을 추가하거나 기존 기능을 변경하려고 할때 사용한다

### 오버라이딩(overriding)

부모 클래스에 있는 메소드를 동일한 이름으로 다시 만드는것을

메소드 오버라이딩이라 합니다

이렇게 메소드를 오버라이딩 하면 부모 클래스의 메소드 대신 오버라이딩한 메소드가 호출됩니다

## 정규 표현식

- 문자 클래스[]

.$*+ 같은 메타문자는

문자 클래스 안에 들어오면 해당의미를 잃고 문자로 취급된다

[]는 그리고 or의 의미를 가진다

- Dot(.)

a.b = a + 모든 문자 하나 + b

aab 나 a0b는 맞지만 abc는 ab가 붙어있음으로 틀리다

a[.]b는 메타문자가 문자 클래스 안에있음으로 a.b문자열과 매치된다

- 반복(*)

ca*t <- 문자 바로 앞에 있는 a가 0번 이상 반복되면 매치

ct 0번 반복되어 매치 cat a가 1번 반복되어 매치 caaat a가 3번 반복되어 매치

- 반복(+)

+는 최소 1번 이상 반복될때 사용한다

cat caaaat은 *와 같이 매치 되지만

위에 *랑 다르게 ct는 0번 반복이라 매치가 되지않는다

- ?

?은 앞에 문자가 1번 이하로 나와야 매치가 된다

- {m}

ca{2}t <- a가 2번 반복되면 매치

caat만 매치된다

- {m,n}

ca{2,5}t <- a가 2~5번 반복되면 매치

cat은 안되고 caat은 되고 caaaaat 은 5번반복이므로 매치

{2,5} {2,} {,5}

"2 ~ 5 "" 2~""    ~5"


| : or

^ : 문자열의 맨 처음 줄 바꿈 하면 줄바꾼 문자열도 검사함

$ : ^와 반대 맨 끝

\A: ^과 동일 하지만 MULTILINE을 옵션을 이용하면 ^는 각 줄의 문자열 처음이지만 A는 줄과 상관없이 전체 문자열의 처음하고만 매치된다

\Z: $과 동일 \A와 같이 멀티라인 옵션을 사용할때는 $ 메타 문자 와는 달리 전체문자열

\b : 단어 구분자(word boundary) 

숫자만 : \d [0~9]

영어대소문자만 : [a~zA~Z]

숫자+영어 : \w [0~9a~zA~Z]

한글만 : [가~힣]
